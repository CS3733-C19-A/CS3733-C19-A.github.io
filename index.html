Alcohol and Tobacco Tax and Trade Bureau

COLA System User Manual

Max Westwater, Jonathan Dang, Shuxing Li, Christopher Bell, Jonathan Tai, James Witt, Dillon Arnold, Junyu Lyu, Alex Rus, Zhihao Xie

Overview

The TTB COLA search application was designed so that manufacturers and agents can better submit and approve alcohol manufacturing forms. This way, not only will productivity of agents be increased but the manufacturers will also be able to easily submit applications for their new projects. The search function has also been improved so that it is much easier to find the information you are looking for, if you are researching or looking up TTB Application forms

Home Page

The Home page contains two majors elements, and serves as the portal from which users will initially interact with the TTB COLA search application

  • COLA Search
    • This button takes you to the COLA Search page
  • Login
    • This button takes you to the Login Page
  • About
    • This button takes you to the About Page
  • Dark Mode
    • This toggle switch turns on and off Dark Mode, which changes the stylesheet of the application to be more suitable to night time usage

Navigation Bar

The Navigation bar is the main way to get around the website. By clicking on the logo you are able to be taken back to the Home Page. When a user is not logged in, the two buttons at the top of the bar are seen to be Login and Signup which take you to the Login and Create Account pages respectively When the a user has logged in then the buttons change to :

  • COLA Search
    • This button takes you to the COLA Search page
  • User Portal
    • This button takes the user to their User Portal
  • Logout
    • This button allows the user to logout of their account
If the user is an Agent they will see an extra button:
  • This button takes users to the User statistics page

COLA Search

The search function is the primary method for user to interact with the TBB Database. Through the COLA Search page, users are able to search for alcohol and view their application information

  • Search Bar
    • The search bar allows you to type in keywords for alcohol that you might be looking for. It is able to suggest results that you might be looking for and pressing the Search button will initiate the search.
    • Next to the search button you are able to choose the type of algorithm you would like to search with, depending on the kinds of results you are looking for
    • The right half of the search bar allows you to download the items that you have searched for in a CSV file format with a chosen Delimiter. You can choose where on your computer you want to save your file to by clicking on the folder icon, and the specific Delimiter in the Delimiter field. Once you are ready, clicking Download results should download the CSV file.
  • Filters
    • The left bar allows a user to filter their search even further through filtering for the category of alcohol and where the alcohol was been sourced from
  • Search Results
    • Search results are displayed in the center of the page and are displayed in infinite scrolling method where more results are loaded as you scroll towards the bottom of the page
    • Clicking on a search result creates an application pop up window which shows all the information of the selected alcohol and its application
